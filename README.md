# PetShop Website

This repository contains the PetShop web app and storefront/admin implementation.

## Quickstart

```bash
cd site
npm i
npm run dev
```

## Baseline HTML Policy

- `site/public` is protected baseline HTML.
- Storefront pages should not be regenerated.
- Admin pages may be edited only when explicitly allowed.

## Shared Copilot Assets

Shared Copilot skills and agent assets were moved to `../copilot-assets`.
For day-to-day work, a multi-root workspace including both repositories is recommended.
